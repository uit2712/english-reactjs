# Install
## Install nodejs
https://nodejs.org/en/download/package-manager
## Install yarn
```bash
npm install -g yarn
```
## Pull code
```bash
git clone https://github.com/uit2712/english-reactjs
```
## Install node_modules
```bash
yarn
```
## Run web
```bash
yarn start
```
# Run tests
```bash
yarn test
```
